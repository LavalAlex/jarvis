# Task 010 — Response Messaging

Stage: 1 — Messaging Infrastructure

---

# Goal

Send a response message back to the user through the WhatsApp API.

---

# Context

Jarvis must respond after receiving and processing a message.

Initially, responses will be simple confirmation messages.

---

# Requirements

1. Implement a service capable of sending messages through WhatsApp.
2. Accept the recipient identifier.
3. Send a text response.

---

# Definition of Done

- Jarvis successfully replies to incoming messages.
