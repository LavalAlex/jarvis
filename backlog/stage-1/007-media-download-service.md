# Task 007 — Media Download Service

Stage: 1 — Messaging Infrastructure

---

# Goal

Implement a service responsible for downloading media files from the WhatsApp API.

---

# Requirements

1. Create a media download service.
2. Retrieve the media URL using the WhatsApp API.
3. Download the file locally.

---

# Definition of Done

- Audio media can be downloaded.
- File path is returned to the caller.
