# Task 004 — Webhook Payload Logging

Stage: 1 — Messaging Infrastructure

---

# Goal

Log incoming webhook events received from WhatsApp.

Logging is essential for debugging and traceability during early development.

---

# Requirements

1. Log the following data:

- message type
- sender identifier
- message timestamp
- message payload

2. Logs must be structured.

3. Logs must not contain sensitive data.

---

# Definition of Done

- Webhook events are logged.
- Logs are readable and structured.
